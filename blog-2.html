Scope

Here in this blog we will be talking about the Scope in JavaScript. We can also term scope as “variable visibility” , scope basically means “The location where a variable is define indicates where we have access to that variable”. Scope is basically divided into three parts which are:-

  1. Block Scope:-

    In the daily use of Javascript we must have used the keyword let/const/var to define a variable , among these three let and const are block scope variable and var have a global scope. This means whenever we define a variable inside a block , (i.e. inside{} ) using let or const, that variable will have a block scope which means it cant be accessed outside those parenthesis, but if the variable is defines using the keyword var then it means that the variable will have a globa scope, lets understand this via examples:-

    Declaring variable using let
    {
    Let x=20;
    }
    Console.log(x); // will show error
    // cannot access x outside the block(parenthesis)
    Declaring variable using var
    {
    Var x=60;
    }
    Console.log(x) // this will show the value of x(i.e. 20) in the console
    // can access the variable outside the scope

  2. Function Scope:-

    Variables which are defined inside a function have function scope, which means those variable cannot be accessed outside the function. Now inside the function it does not means weather the declare a variable using let/const or var it will have a function scope only

    variable defined using let
    Function temp(){
    Let X=20; // cant be accessed outside
    }
    Console.log(x) // will show error regarding x not defined
    variable defined using var
    Function temp(){
    Var x=90; // cant be accessed outside the function
    }
    Console.log(x) // this will show an error regarding x not defined

  3. Global Scope:-

    Global scope means the variables which are defined globally , not inside any function or not inside any block have a global scope, which means it can be accessed anywhere in the whole program doesn’t if the variable is defined using let/const or var it ca be used anywhere.
